  • {{ whats-new-list-1 Improved ad blocking. Many websites look for ways to prevent Adblock Plus from blocking annoying ads. If you ever run into a website that won’t allow you to block ads, please let us know by using our Report Issue feature. }}
  • {{ whats-new-list-2 Added Smart Language Filters. When you turn on this feature, you’ll be automatically notified of available filter lists in the languages you regularly browse in.   • {{ whats-new-list-3 Added support for blocking ads in more languages. Users who browse in Finnish and Swedish can now block ads on websites that are using these languages. }}
  • {{ whats-new-list-4 Improved user experience. We’re always at work to make Adblock Plus better. We’ve recently updated several error messages for easier debugging, made changes to our terminology, and added a new footer to the pop-up menu so you can conveniently access what you need. }}
