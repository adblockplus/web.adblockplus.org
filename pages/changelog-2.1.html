title=Detailed changelog for Adblock Plus 2.1

{{s1 The following lists the major changes compared to Adblock Plus 2.0.3.}}

{{s2 Features}}

  • {{s3 Adblock Plus can be installed without restarting the browser now.}}
  • {{s4 Added support for Firefox Mobile 14.}}
  • {{s5 Better Thunderbird integration, in particular the Adblock Plus user interface is working with web page tabs correctly now.}}
  • {{s6 Implemented element hiding exceptions (mostly useful for subscription authors).}}

{{s7 User interface}}

  • {{s8 Filter composer: Removed warning when adding filters to a disabled group, a new group is created if no enabled group can be found for the filter.}}
  • {{s9 Filter preferences: Various improvements for a better overview (forum topic, forum topic, forum topic).}}
  • {{s10 Filter preferences: Fixed wrong warning message when deleting filter groups.}}
  • {{s11 Issue reporter: Force a filter subscription update when reporting a false positive (forum topic).}}
  • {{s12 Issue reporter: Sending issue reports is no longer allowed if the filter subscription hasn't been updated in the past two weeks.}}
  • {{s13 Custom About dialog has been removed, using the browser's built-in dialog now.}}
  • {{s14 Fixed: Tabs on Flash and Java no longer have rounded corners in Firefox 13 (forum topic).}}

{{s15 Blockable items}}

  • {{s16 The shortcut key for Blockable items in Firefox 15 and above is Ctrl-Shift-U now, the old shortcut key is no longer available.}}
  • {{s17 Searches for object-subrequest are accepted as well, searching for object_subrequest is no longer required (forum topic).}}
  • {{s18 Fixed: The list will sometimes reload even though the current page stays the same.}}
  • {{s19 Fixed: Context menu might suggest disabling the filter on wrong domain (forum topic).}}
  • {{s20 Fixed: Context menu might prevent page elements from being garbage collected (bug 727739).}}
  • {{s21 Fixed: The tooltip might get displayed on top of the context menu.}}

{{s22 Core}}

  • {{s23 Adblock Plus can now be installed in any application based on Gecko 8.0 and higher, in particular Firefox 8, Thunderbird 8, SeaMonkey 2.5. Older versions are no longer supported, XUL-based Firefox Mobile releases like Firefox Mobile 10 aren't supported either.}}
  • {{s24 Filter subscriptions can set their title automatically using ! Title comment (forum topic).}}
  • {{s25 Improved memory use and performance, especially fixed the regression in Firefox 15 and above.}}
  • {{s26 Stopped using cache.js file, it's a frequent source of stability issues.}}
  • {{s27 Switched to asynchronous file read/write operations to improve user interface responsiveness.}}
  • {{s28 Fixed: Object tabs are causing "can't access dead object" message in Error Console of Firefox 15 and above.}}
  • {{s29 Fixed handling of trailing dots in domain names (forum topic).}}
