title=Using Adblock Plus in K-Meleon

{{s1 Installation}}

{{s2 Note: The following Adblock Plus versions will only work in K-Meleon 1.6. To install you need to download a special ZIP package:}}

{{s3 Download Adblock Plus}}

{{s4 Unpack this package with any ZIP unpacker into the directory of your K-Meleon browser, all files will be placed into the right directories. Then start K-Meleon, it will ask you whether Adblock Plus plugin should be activated — confirm this. That's all, Adblock Plus should work now.}}

{{s5 For K-Meleon 1.5 you can download Adblock Plus 1.0.2.}}

{{s6 Uninstalling Adblock Plus}}

{{s7 Note that deactivating Adblock Plus plugin will only remove it from the user interface, usually the extension will continue blocking in the background. To uninstall Adblock Plus you need to remove all the files you installed with the ZIP package.}}

{{s8 Known differences to Firefox}}

{{s9 Adblock Plus currently has a number of limitations in K-Meleon if compared to the Firefox version. The following ones might be fixed by future Adblock Plus versions with more complete K-Meleon support:}}

  • {{s10 ABP icon state and list of blockable items do not update correctly if a blank tab is open}}
  • {{s11 Blockable items must stay detached, K-Meleon’s sidebar implementation isn't yet supported}}
  • {{s12 Blockable items window cannot be closed with hotkeys (Esc or Ctrl+Shift+V)}}
  • {{s13 Preferences dialog: "Show in toolbar" option should be hidden}}

{{s14 There are other limitations that are rooted in K-Meleon's architecture and are unlikely to be fixed unless changes are made to K-Meleon:}}

  • {{s15 Blockable items window is always on top and refers to all browser windows}}
  • {{s16 Clicking report link after sending off an issue report causes a crash}}
  • {{s17 Size of the "Add filter subscription" dialog changes incorrectly when switching to "Add a different subscription" and back}}
  • {{s18 Status bar icon doesn't have a tooltip}}
  • {{s19 Toolbar icon cannot be hiden via Adblock Plus options, only via View / Toolbars menu}}
  • {{s21 Shortcut key preferences like extensions.adblockplus.sidebar_key use Win32 virtual key codes instead of Gecko’s virtual key codes}}
