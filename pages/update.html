{{ whats-new-header-1 What’s new? }}

{{ whats-new-lead-1 Here is an overview of some improvements and new features. More changes coming soon! }}

  • {{ whats-new-list-1 New Recommended Filters. We’ve made it easier to block additional distractions such as cookie notifications and more. Visit Settings > General > Recommended filters to try them out! }}
  • {{ whats-new-list-2 Improved Ad Blocking. Many websites make it difficult for Adblock Plus to block ads on their site. We’ve improved our ability to block ads on these websites, and update this ability regularly. Please use our Report Issue feature to let us know if you have any issues with our ad blocking ability. }}
  • {{ whats-new-list-3 Better Performance. We’ve improved our logic when checking for turned off filter lists so that Adblock Plus uses less bandwidth. }}
  • {{ whats-new-list-4 Improved Privacy. For our users browsing in private tabs, we’ve excluded the number of blocked requests from the total blocked counter. }}
  • {{ whats-new-list-5 Better Debugging. For our technical users we introduced a debug mode for element-hiding filters that can be turned on in the Adblock Plus Advanced settings page. You can easily highlight elements that are affected by element-hiding filters. }}
