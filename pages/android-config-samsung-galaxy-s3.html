title=Proxy configuration on Samsung Galaxy S III
  1. {{s1 Run Settings application.}}
  2. {{s2 Select Wi-Fi:}}
  3. {{s3 Connect to the desired network (access point) if not already:}}
  4. {{s4 Long press on connected network, context menu will appear:}}
  5. {{s5 Select Modify network config.}}
  6. {{s6 Mark Show advanced options check box:}}
  7. {{s7 Scroll to proxy settings, select Manual, proxy options will appear:}}
  8. {{s8 Set Proxy to localhost and Port to the number specified by Adblock Plus in the notification message:}}
