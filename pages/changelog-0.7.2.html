title=Detailed changelog for Adblock Plus 0.7.2

{{s1 The following lists the major changes compared to Adblock Plus 0.7.1:}}

{{s2 New features}}

  • {{s3 Thunderbird and SeaMonkey Mail are now supported (bug 13008)}}
  • {{s4 K-Meleon 1.02 and higher is now supported with some limitations (installation for K-Meleon)}}
  • {{s5 Installation in Songbird now possible (no user interface integration yet, users should install a subscription via Adblock Plus preferences which can be opened with File / Extensions / Options)}}

{{s6 User interface}}

  • {{s7 Added translations: Slovenian, Hungarian, Vietnamese}}
  • {{s8 Switched to new icon set (forum thread)}}
  • {{s9 Welcome dialog only allows to add one subscription now (forum thread)}}
  • {{s10 Only the really universal and recommendable subscriptions are now displayed in the welcome dialog, others can be installed from the web page the dialog links to (forum thread)}}
  • {{s11 Fixed invisible checkboxes in welcome dialog with some Linux distributions (forum thread)}}
  • {{s12 Fixed item width in welcome dialog in SeaMonkey}}

{{s13 Core}}

  • {{s14 New pattern matching algorithm speeds up processing of large filter lists even more (blog entry)}}
  • {{s15 Object tabs now have minimal impact on the page's HTML code, this resolves problems with WYSIWYG editors — especially the ones used by blog publishing software (blog entry)}}
  • {{s16 Several parts of the extension should execute faster now}}
  • {{s17 Content loaded from keyword: URLs can now be blocked as well (bug 15072)}}
  • {{s18 Removed URL validation when deciding whether a URL scheme is allowlisted (bug 15072)}}
  • {{s19 Made about:blank an exception, it isn't allowlisted by its scheme any more (bug 14793)}}

{{s20 Known issues}}

  • {{s21 Yahoo Application State plugin (npYState.dll, installed by Yahoo Messenger) causes crashes on Yahoo Mail if Adblock Plus is installed. It is suggested to remove this plugin, its only function seems to be to notify Yahoo Mail about your version of Yahoo Messenger.}}
  • {{s22 Danish, Korean and Traditional Chinese translations contain a few English strings}}
  • {{s23 Status bar icon is always moved to the very end of the status bar, not even Organize Status Bar can change its position}}
  • {{s24 Thunderbird/SeaMonkey Mail: Default keyboard shortcut Ctrl+Shift+A for Adblock Plus Preferences doesn't work due to a conflict with a default keyboard shortcut (Edit / Select / Thread)}}
  • {{s25 Thunderbird/SeaMonkey Mail: Blockable items list is always detached — sidebars are not supported by the application}}
  • {{s26 SeaMonkey Mail: No toolbar icon available (status bar icon shows up instead)}}
