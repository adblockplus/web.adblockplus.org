title=Detailed changelog for Adblock Plus 2.0

{{s1 The following lists the major changes compared to Adblock Plus 1.3.10.}}

{{s2 Features}}

  • {{s3 Experimental pop-up blocking support for the cases where the built-in pop-up blocker fails (forum topic).}}
  • {{s4 New Filter Preferences dialog makes managing filters and subscriptions easier (forum topic).}}
  • {{s5 Custom filters can be organized into any number of filter groups.}}
  • {{s6 Backup and Restore functionality allows creating a copy of all filter preferences, not just custom filters.}}
  • {{s7 Added "Allow nonintrusive advertising" setting (can be changed in Filter Preferences, documentation).}}

{{s8 User interface}}

  • {{s9 Lithuanian translation is included again.}}
  • {{s10 Adblock Plus options display directly on its page in the Add-ons Manager.}}
  • {{s11 "Recommend us on Facebook" button has been replaced by a more generic "Contribute".}}
  • {{s12 Blockable items: Added "Filter source" column (hidden by default, forum topic).}}
  • {{s13 Blockable items: Searching by type now works both for localized type name as well as corresponding filter option (forum topic).}}
  • {{s14 Filter composer: Filter options are now added in the order in which they are selected (forum topic).}}
  • {{s15 Filter preferences: A warning is displayed for subscriptions containing disabled filters (forum topic).}}
  • {{s16 Issue reporter: Subscriptions that are local files aren't sent to server, for privacy reasons.}}
  • {{s17 Issue reporter: Fixed check that the user is subscribed to a filter subscription (forum topic).}}
  • {{s18 Simplified options for showing the Adblock Plus icon and made them work more consistently.}}
  • {{s19 "Count filter hits" is now a visible option.}}
  • {{s20 "Collapse blocked elements" option has been renamed into "Hide placeholders of blocked elements".}}
  • {{s21 "Adblock Plus" entry in the Tools menu is now the same menu as displayed for the Adblock Plus icon for accessibility reasons, the menu entry in the View menu has been removed.}}
  • {{s22 Fixed: Toolbar button's context menu opens when right-clicking the menu itself (forum topic).}}
  • {{s23 Fixed: Using context menu prevents a memory compartment from being released (forum topic).}}

{{s24 Core}}

  • {{s25 Minimal required Firefox version is now Firefox 3.6.12, older versions are no longer supported. Similarly, at least Thunderbird 3.1.7 or SeaMonkey 2.1 are required, Songbird and Prism are no longer supported.}}
  • {{s26 document and elemhide filter options for exception rules can now apply to frames as well, not only at the top level.}}
  • {{s27 Added sitekey filter option to allow exceptions that apply to multiple websites.}}
  • {{s28 Filter options xbl, dtd and ping are now considered synonymous to other, the corresponding requests are very rare (forum topic).}}
  • {{s29 Simplified redirect tracking, removed a common source of memory issues and other problems (bug 672111, bug 653533).}}
  • {{s30 Comment ! Homepage: http://example.com/ in filter subscriptions is now being recognized.}}
