title=Source Code

{{s1 The Adblock Plus project consists of multiple repositories.}} {{s2 The modules page has a good overview of the most important ones.}}

{{s3 Getting the source code from the Mercurial repositories}}

{{s4 The Adblock Plus source code is hosted on our Mercurial server.}} {{s5 You can see a list of all repositories through the web interface.}}

{{s6 Here's how you clone the Adblock Plus for Firefox repository:}}

hg clone https://hg.adblockplus.org/adblockplus/

{{s7 Here's how you get the source code for a specific version (e.g. Adblock Plus 2.3.1):}}

hg update -r 2.3.1

{{s13 Getting the source code from the GitHub mirrors}}

{{s14 If you prefer to use Git, you can - we have GitHub mirrors for all our public repositories.}}

{{s15 Here's how you clone the Adblock Plus for Firefox repository from GitHub:}}

git clone https://github.com/adblockplus/adblockplus

{{s16 Here's how you get the source code for a specific version (e.g.}} Adblock Plus 2.3.1{{s17 ):}}

git checkout 2.3.1

{{s18 Please note: We are currently using Mercurial subrepositories heavily, but are unable to mirror these as Git submodules.}} {{s19 That means that you need to check the .hgsub file of each repository and clone all dependencies manually.}} {{s20 This won't be necessary anymore once we've moved away from subrepositories.}}
