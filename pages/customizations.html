title=Customizations for Adblock Plus

{{s1 General notes}}

{{s2 The user interface of Adblock Plus is intentionally not very customizable, the idea is to have it work well out of the box without requiring users to tweak things.}} {{s3 As a result, some compromises have to be made and sometimes the user interface is suboptimal for power users.}} {{s4 Customizations extension is a collection of semi-supported user interface tweaks for people who feel the need to change something.}}

{{s5 Installation}}

{{s6 Install Customizations}}

{{s7 How to use}}

{{s8 Customizations can be installed and updated without restarting the browser.}} {{s9 After installing you can go to the Add-ons Manager and press the Options button for this extension.}} {{s10 You will see the add-on description followed by a list of available options.}} {{s11 Example options:}}

  • {{s12 Various Filter Preferences tweaks to create a minimalistic layout}}
  • {{s13 Toolbar icon display (text only, image only, both)}}
  • {{s14 Remove Adblock Plus from all browser menus}}
  • {{s15 Tweaking some hidden Adblock Plus preferences}}
